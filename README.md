# ZIPbrute
ZIP加密文件密码破解器
~~~~~~~使用字典暴力枚举~~~~~~~~

2022.1.21     v1.0
使用模板：
请输入你要爆破的压缩文件，例：C:\a.zip
>>>{要爆破的目标文件路径}
请输入要使用的字典文件，输入0则使用默认字典
>>>{字典文件}
Found Passwd :  爆破出的密码
Found Passwd :  爆破出的密码
Password =  爆破出的密码
